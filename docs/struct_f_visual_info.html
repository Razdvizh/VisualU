Detailed Description

Base struct for Visual Info system.

Represents certain data which other classes can retrieve through IInfoAssignable interface. To create custom Visual Info struct:

  1. Create a struct that inherits from FVisualInfo
  2. Add desired fields
  3. Override Accept function to implement Visitor pattern (see FVisualImageInfo::Acept for example)
  4. Override ToString function if needed Check FVisualImageInfo
