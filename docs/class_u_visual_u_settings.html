Detailed Description

Global plugin settings.

You can find them under "Plugins" settings category.

The class is derived directly from UObject and simply contains necessary data. It is not a subclass of UDeveloperSettings due to the internal structure of the Unreal plugins. Registration for engine's settings system occurs through ISettingsModule inside Module struct.

TSoftObjectPtr<UDataTable> UVisualUSettings::FirstDataTable

Data Table that represents the first branch of scenarios when game starts.

Contains the first scenario that will be loaded and displayed by Visual Scene; the entry point in the VisualU system. Visual Scene handles the processing and constructing of the first scenario, if the Data Table is empty, the assertion will be triggered.
