Detailed Description

Supports Visual Novel style of typewriter effect.

Visual Text Block can be used as Rich Text Block for static rich text and embedded images, fields that Visual Text Block provides control only typewriter effect. Typewriter mode is achived by calling UVisualTextBlock::SetText method. Use of embedded images in this mode may lead to unexpected behavior, other features that Rich Text Block has are supported.
