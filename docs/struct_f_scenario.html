Detailed Description

A single scene or "frame" of the Visual Novel game.

Struct with data that describe one scene handled by Visual Scene. Can be imported from CSV or JSON file or added directly as a row to the Data Table that has FScenario row struct. Terms "Scene", "Row" and "Scenario" all refer to FScenario. Visual Scene and "Scene" are distinguished by word "Visual" in front of the class one. Grouped together under Data Table, they represent a node of the tree structure that allows to create multiple ways to complete the game. The first scene in the game is specified by UVisualUSettings::FirstDataTable field. The last scene in the node can have Visual Choice sprite that links nodees together. Connections between scenes are managed by Visual Scene. That being said, FScenario is an intrusive data structure, meaning that FScenario is well aware in what node and where in the node it resides. That implies that FScenario must be a part of a node and cannot be owned by some other entity. FScenario is a polymorphic struct and can be extended if needed.
