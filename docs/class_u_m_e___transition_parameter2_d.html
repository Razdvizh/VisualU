Detailed Description

Material node for transition materials.

Represents either first (currently rendered), or second (final target) sprite in the material asset which serves as transition effect. Parameters are immutable in the node and can only be switched by UME_TransitionParameter2D::bIsSecondTransition. Parameter names, however can be changed in the Visual Settings for all transition parameter nodes.
