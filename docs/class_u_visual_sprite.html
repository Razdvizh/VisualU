Detailed Description

Base class for widgets that represent sprites that will be visualized by Visual Scene.

Visual form of the classic Visual Novel sprite. Can represent NPC, an interactive object, etc. Visual Sprite contains any amount of Visual Images needed to display a sprite, which can be altered by Visual Scene using information provided in the different Scenarios. It can contain any functionality of the normal widget and can be extended as some specific kind of the sprite. The arrangement of Visual Images is meant to be done inside blueprint class.

Default implementation will assign first FVisualImageInfo to the first Visual Image, which is the one closest to the root of the WidgetTree.
